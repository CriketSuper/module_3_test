# Проверка module_3

Скрипт запускается на HQ-CLI от `root`. Проверьте адреса, пароли и ожидаемые параметры в `.env`.

```bash
cd module_3_test
chmod +x 01-check-module-3.sh
./01-check-module-3.sh
```

Проверяются пункты `1-9` и `11`. Пункты `10`, `12` и `13` выводятся как `N/A`, итог рассчитывается из 10 баллов.

Проверка fail2ban выполняет три ошибочных SSH-входа, проверяет блокировку и снимает её. Отключить активный тест можно параметром:

```bash
ACTIVE_FAIL2BAN_TEST=no
```

Проверить только формат таблицы:

```bash
./01-check-module-3.sh --self-test
```
